## 特征向量
### 定义
给定$Ax$,$x$作用于方阵$A$后得到的向量如果仍旧平行于$x$[[矩阵的视角]]
即核心方程：$$Ax = \lambda x$$
**特殊的向量**，它们在变换后**方向保持不变**。这些就是特征向量。 ^5e1c29
- $\lambda$也被称作“特殊值”

#### 特征值的特殊情况
#### 特征值为零
此时特征向量为零空间中的向量，满足：[[向量空间及其子空间#^0ed6c9|特征向量张成了零空间]]
$$
Ax=0
$$
**$A$作用到向量$x$后得到零**
此时的矩阵$A$奇异矩阵(singular)[[singular矩阵]]

## 投影矩阵的特征
### 第一类特征向量
如果向量$x$属于矩阵的列空间，投影满足：[[投影、投影矩阵#^a956d5|向量的投影]]
$$Px = 1 \cdot x$$
- **特征向量：** 所有位于投影平面内的向量。
- **特征值：** $\lambda = 1$。

### **第二类特征向量**
如果向量$x$垂直于平面，属于矩阵的左零空间,满足:
$$
Px = 0 \cdot x
$$
- **特征向量：** 所有垂直于投影平面的向量。
- **特征值：** $\lambda = 0$。

### 特征值的数学解释
$P^2 = P$，且 $Px = \lambda x$，代入得：$$\lambda x = \lambda (\lambda x)$$$$\lambda x = \lambda^2 x$$消去向量 $x$（因为 $x \neq 0$）：$$\lambda = \lambda^2$$
## 特征值的性质
### 特征值的个数
$n\times n$矩阵有$n$个特征值（如果特征值不相等，就有$n$个线性无关的特征向量，**特征向量矩阵可逆**）[[对角化和矩阵乘幂#^2d96b0|特征向量矩阵]]

### 特征值与迹
特征值之和等于方阵对角线元素之和，即方阵的迹（trace）：- $$\sum \lambda_i = \text{Trace}(A)$$

### 特征方程
对于矩阵方程$Ax = \lambda x$，它移项变成：

$$(A - \lambda I)x = 0$$

想要找非零向量 $x$，矩阵 $(A - \lambda I)$ 必须是奇异矩阵（singular）。[[singular矩阵|矩阵不满秩]]
奇异矩阵的行列式必为 0，得到**特征方程：**
$$\det(A - \lambda I) = 0$$
### 迹和行列式
对于二阶矩阵，特征方程可以表示为：$$\lambda^2 - (\text{Trace})\lambda + (\text{Determinant}) = 0$$
- $\lambda_1 + \lambda_2 = \text{Trace}$  
- $\lambda_1 \cdot \lambda_2 = \text{Determinant}$                                                                                                                    

#### 推广到 $n \times n$ 矩阵
特征多项式 $P(\lambda) = \det(A - \lambda I)$，展开成 $\lambda$ 的多项式：
$$P(\lambda) = (-1)^n \lambda^n + c_{n-1} \lambda^{n-1} + \dots + c_1 \lambda + c_0 = 0$$
##### 次高次项（$\lambda^{n-1}$）的系数
它总是与 **迹（Trace）** 有关。
- 系数 $c_{n-1} = (-1)^{n-1} \times \text{Trace}(A)$
- 这解释了为什么 **特征值之和 = 迹**。

##### 常数项（$c_0$）
它总是等于 **行列式（Determinant）**。
- 系数 $c_0 = \det(A)$
- 这解释了为什么 **特征值之积 = 行列式**。
##### 原因
$P(\lambda) = \det(A - \lambda I)$ 中，令 $\lambda = 0$：
$$P(0) = \det(A - 0 \cdot I) = \det(A)$$
而在多项式 $c_n \lambda^n + \dots + c_0$ 中，令 $\lambda=0$，剩下的就是 常数项 $c_0$。
所以，常数项必然等于 $\det(A)$。

### 矩阵的平移
矩阵加上单位矩阵，对特征值和特征向量的影响：
1. **特征向量（Eigenvectors）：完全不变**。
2. **特征值（Eigenvalues）：全部 +1**。
#### 证明
$$(A + I)x = Ax + Ix$$
因为 $Ax = \lambda x$ 且 $Ix = x$，代入得：
$$= \lambda x + x$$

提取公因式 $x$：
$$= (\lambda + 1)x$$
## 应用
1. 判断可逆性（Invertibility）：
    如果矩阵 $A$ 有一个特征值是 $0$，那么 $A$ 是不可逆的（奇异的）。[[singular矩阵]]
    但是，$A + I$ 的特征值就是 $0+1=1$，它变可逆了！这在数值计算算法中非常重要（Google 的 PageRank 算法）
2. 求解微分方程：
    当解 $\frac{du}{dt} = (A+I)u$ 时，因为你知道特征向量没变，解的结构保持不变，只是指数上的增长率从 $e^{\lambda t}$ 变成了 $e^{(\lambda+1)t}$。

### 适用范围
**只**适用于$A + cI$。 如果你计算 $A + B$（两个普通矩阵相加），**特征值通常不会相加** ($\lambda_A + \lambda_B \neq \lambda_{A+B}$​)，特征向量也通常会完全改变。只有当 A 和 B 拥有**相同的特征向量**时，才会有简单的规律。

## 复数特征值
复数特征值意味着矩阵含有“旋转（Rotation）”成分，或者说矩阵是“非对称（Asymmetric）”
### 旋转矩阵 (Rotation Matrices)
 $Ax = \lambda x$ 的几何意义是“向量 $x$ 在变换后方向不变”，对于旋转 90° 的矩阵：

$$Q = \begin{bmatrix} 0 & -1 \\ 1 & 0 \end{bmatrix}$$
没有实向量能满足“方向不变”，那么就不存在实数特征值。
### 反对称矩阵 (Skew-Symmetric Matrices)
- **对称矩阵 ($A = A^T$)：** 只有**实数**特征值。（这是好矩阵）
- **反对称矩阵 ($A = -A^T$)：** 只有**纯虚数**特征值（Pure Imaginary）。
### 复数特征值特点
共轭成对出现 (Conjugate Pairs)
因为矩阵 $A$ 里的元素都是实数，特征多项式的系数也是实数。根据代数基本定理，实系数多项式的复数根必然是共轭成对的。
