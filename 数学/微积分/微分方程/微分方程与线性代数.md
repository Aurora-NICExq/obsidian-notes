理论基础：**无穷维向量空间中的线性代数**。
## 核心映射

|**概念**|**线性代数 (Linear Algebra)**|**线性微分方程 (Linear Diff Eq)**|
|---|---|---|
|**基本方程**|$A\mathbf{x} = \mathbf{b}$|$L[y] = g(x)$|
|**算子/变换**|矩阵 $A$ (Matrix)|微分算子 $L$ (如 $\frac{d^2}{dx^2} + p\frac{d}{dx} + q$)|
|**未知量**|向量 $\mathbf{x}$|函数 $y(x)$|
|**齐次方程**|$A\mathbf{x} = \mathbf{0}$|$L[y] = 0$ (即 $y'' + py' + qy = 0$)|
|**非齐次项**|向量 $\mathbf{b}$|函数 $g(x)$ (非齐次项/驱动力)|
## 通解的结构
解的结构公式是**完全一致**的：[[向量空间及其子空间#^981165|通解与特解的关系]]
$$\text{通解} = \text{齐次通解} + \text{非齐次特解}$$
###  微分方程视角

对于方程 $y'' + p(t)y' + q(t)y = g(t)$，其通解结构为：
$$y(t) = y_h(t) + y_p(t)$$

- **$y_h(t)$ (齐次通解):** 它是 $y'' + py' + qy = 0$ 的解。通常由两个线性无关的解线性组合而成：$C_1y_1 + C_2y_2$。
- **$y_p(t)$ (特解):** 是非齐次方程的一个特定解。
**结论：** 在求微分方程的通解时，实际上是在寻找微分算子 $L$ 的“零空间”（Kernel），然后加上一个偏移量（特解）。
## 线性无关与基底 (Basis)
- **线性代数：** 如果零空间的维数是 $n$，需要找到 $n$ 个**线性无关**的向量 $\mathbf{v}_1, \dots, \mathbf{v}_n$ 作为基底。
    - 齐次通解：$c_1\mathbf{v}_1 + \dots + c_n\mathbf{v}_n$
    - 判定无关性：使用矩阵的秩 (Rank) 或行列式。 [[向量组的性质#^8403a0|线性代数的基]]
- **微分方程：** 如果是 $n$ 阶线性微分方程，解空间的维数通常也是 $n$，需要找到 $n$ 个**线性无关**的函数 $y_1, \dots, y_n$。
    - 齐次通解：$c_1 y_1(x) + \dots + c_n y_n(x)$
    - 判定无关性：使用 **朗斯基行列式 (Wronskian)**。
    - _注：朗斯基行列式本质上就是在检查这些函数构成的“向量”是否线性无关。_

## 微分方程与线性代数的共同点
求导 ($\frac{d}{dx}$) 本身就是一个线性运算。
这就意味着它满足线性的叠加原理：
$$L[c_1 y_1 + c_2 y_2] = c_1 L[y_1] + c_2 L[y_2]$$
正是因为微分算子是线性的，才能像拆解向量一样，把复杂的方程拆解为“齐次部分”和“特解部分”分别处理，最后再叠加在一起。
