| 若 $f$ 是一个                           | 则形式为 ($y_P$)                                  |
| :---------------------------------- | :-------------------------------------------- |
| **次数为 $n$ 的多项式**                    | **$y_P =$ 次数为 $n$ 的一般多项式**                    |
| 例：$f(x)=7$                          | $y_P = a$                                     |
| $f(x)=3x-2$                         | $y_P = ax+b$                                  |
| $f(x)=10x^2$                        | $y_P = ax^2+bx+c$                             |
| $f(x)=-x^3-x^2+x+22$                | $y_P = ax^3+bx^2+cx+d$                        |
| **指数 $\mathrm{e}^{kx}$ 的倍数**        | **$y_P = C\mathrm{e}^{kx}$**                  |
| 例：$f(x)=10\mathrm{e}^{-4x}$         | $y_P = C\mathrm{e}^{-4x}$                     |
| $f(x)=\mathrm{e}^{x}$               | $y_P = C\mathrm{e}^{x}$                       |
| **$\cos(kx)$ 的倍数 + $\sin(kx)$ 的倍数** | **$y_P = C\cos(kx) + D\sin(kx)$**             |
| 例：$f(x)=2\sin(3x)-5\cos(3x)$        | $y_P = C\cos(3x) + D\sin(3x)$                 |
| $f(x)=\cos(x)$                      | $y_P = C\cos(x) + D\sin(x)$                   |
| $f(x)=2\sin(11x)$                   | $y_P = C\cos(11x) + D\sin(11x)$               |
| **上面某些形式的和或积**                      | **这些形式的和或积 (若为积，删掉一个常数)**                     |
| 例：$f(x)=2x^2 + \mathrm{e}^{-6x}$    | $y_P = ax^2+bx+c + C\mathrm{e}^{-6x}$         |
| $f(x)=2x^2\mathrm{e}^{-6x}$         | $y_P = (ax^2+bx+c)\mathrm{e}^{-6x}$           |
| $f(x)=7\mathrm{e}^{2x}\sin(3x)$     | $y_P = (C\cos(3x)+D\sin(3x))\mathrm{e}^{2x}$  |
| $f(x)=\cos(2x)+6\sin(x)$            | $y_P = C\cos(2x)+D\sin(2x)+E\cos(x)+F\sin(x)$ |
| $f(x)=4x\cos(3x)$                   | $y_P = (x+b)(C\cos(3x)+D\sin(3x))$            |

> **⚠️修正规则**：若 $y_P$ 与 $y_H$（齐次通解）中的项冲突（线性相关），令特解的形式乘以 $x$ 或 $x^2$。
