**将一个难以直接计算的积分，转化（交换）为一个更容易计算的积分。**

## 分部积分原理
分部积分的本质来源于导数的乘法法则。
乘法公式：$$d(uv) = u\,dv + v\,du$$
对两边同时积分并移项，得到分部积分公式：
$$\int u \, dv = uv - \int v \, du$$
### 变量的选择
#### LIATE 原则
为了保证 $\int v \, du$ 变简单，按照以下优先级**从上到下**选择 $u$（剩下的部分就是 $dv$）：
1. **L (Logarithmic)**：对数函数（如 $\ln x$）
2. **I (Inverse trigonometric)**：反三角函数（如 $\arctan x$, $\arcsin x$）
3. **A (Algebraic)**：代数/幂函数（如 $x^2$, $3x$, $x^n$）
4. **T (Trigonometric)**：三角函数（如 $\sin x$, $\cos x$）
5. **E (Exponential)**：指数函数（如 $e^x$）

$dv$选择容易积分的函数，$u$选择求导后不复杂的函数

### 分部积分示例
计算：
$$\int x \cos x \, dx$$
进行交换
$$\int x \cos x \, dx = x \sin x - \int \sin x \, dx$$
最终得到：$$ x \sin x + \cos x + C$$
### 特殊情况
#### 循环积分
$\int e^x \sin x \, dx$

#### 表格法
$\int x^3 e^x \, dx$
