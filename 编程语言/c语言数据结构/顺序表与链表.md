## 线性表
1. 定义：线性表是 $n$ 个数据元素的有限序列，其中 $n$ 个数据是相同数据类型的。
2. 特点：有头有尾
     - 除第一个元素外，结构中的每个数据元素均只有一个前驱
     - 除最后一个元素外，结构中的每个数据元素均只有一个后继
3. 存储结构
```c
#define MAXSIZE 100
typedef int ElemType;  //宏定义，提高代码维护性

typedef struct{
    ElemType data[MAXSIZE];
    int length;
}SeqList;

void Initlist(SeqList* L){
	L->length=0;  //传指针节省空间
}
```
4.顺序表尾部添加元素
```c
//首先判断顺序表未满
L->data[L->length]=e;
L->length++;
return 1;   //表示成功
```
