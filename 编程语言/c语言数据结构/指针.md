## 指针的表示
1. 定义：储存地址的变量
2. 指针的数据类型
	- int,char....
3. 指针的赋值
	- 野指针（指针不指向任何变量）
	- 空指针：指针指向NULL
4. 解引用运算符：返回指针变量指向的地址的值
```c
int a=5;
int* p=&a;   //p指向a的地址
printf("%d",*p); //解引用指针
*p=100;
printf("%d",a);  //此时a=100
```


## 数组和指针的关系
通过数组下标能完成的操作都可以用指针完成
   数组名就是首元素的地址（指针常量）:
```c
	  int a[]={10,11,12,13};
	  int* p;
	  p=a;
	  printf("%p\n",a); //输出a[0]的地址
	  printf("%p\n",p); //p=a[0] (地址)
	  printf("%\n",*p);
```

```c
c[0]. *c
c[1]  *(c+1);
```

### 数组的实质
1. 数组是一些相同数据类型的变量组成的集合，数组名为该数组类型的指针
2. 数组的定义等效于申请内存，定义指针和初始化
```c
char c[]={0x33,0x34,0x35};
//申请内存
//定义 char* c=0x4000;
//初始化数组数据
```



## 指针的算术运算
### 指针的加减
1. 加上一个整数，加的是这个整数的数据类型所对应的字节数的积（指针变量有数据类型的原因）
2. 指针的加减只在连续内存中有效
```c
*p=c+1. //表示p指向c之后的地址
```
