### 算术操作符
1. 类型： + - * / %
2. 除法分类
	- 整型除法：1/2->0
	- 浮点型除法：1.0/2->0.5     （需要保证除号两端至少一个数是浮点数）
	- 取余运算符%：7%2->3        (两端必须是整数)
3. 移位操作符   （操作数只能是整数）
	- 分类：<<左移操作符
	        >>右移操作符
	
```c
	int first=7;
	int second=first<<1;   //second=14
```
4. 整数的二进制表示
	- 表示形式：原码，补码，反码
	- 正的整数三种均相同
	- 负的整数需要计算
	- 最高位是符号位：正数最高位为0，负数最高位为1
	- 补码：反码+1
	- 整数在内存中存储的是补码
5. 左移操作符特点
	- <<移动补码
	- 计算规则：左边丢弃，右边补零

6. 右移操作符特点
	- 算术移位：右边丢弃，左边补原符号位
	- 逻辑移位：右边丢弃，左边补零
