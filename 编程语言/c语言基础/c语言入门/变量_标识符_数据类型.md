## 标识符
C 语言中用来给变量、常量、函数等内容起名字时用的字符序列
标识符的命名规则如下：

- 标识符由字母、数字、下划线（_）组成，且必须以字母下划线开头
    
- 标识符对大小写敏感，也就是说，abc、Abc和ABC被视为三个不同的标识符
    
- 不能使用 C 语言的关键字作为标识符

### 标识符规范
```c
char user_name1 //下划线命名
char userName  //驼峰标识
```

## 数据类型
### 整形
```c
int user=100;%
printf("%d",user) //%d表示十进制整数
```
**通配符，占位符一一对应变量名**

| 输出格式通配符（占位符） | 意义          |
| :----------- | :---------- |
| %d           | 十进制整数       |
| %o           | 八进制整数       |
| %x           | 十六进制整数，字母小写 |
| %X           | 十六进制数，字母大写  |
| %u           | 十进制无符号数     |
### 整形类型

| 数据类型               | 占内存空间                   | 通配符  |
| :----------------- | :---------------------- | :--- |
| short              | 2字节                     | %hd  |
| int                | 4字节                     | %d   |
| long               | Windows为4字节，Linux为4或8字节 | %ld  |
| long long          | 8字节                     | %lld |
| unsigned short     | 2字节                     | %hu  |
| unsigned int       | 4字节                     | %u   |
| unsigned long      | Windows为4字节，Linux为4或8字节 | %lu  |
| unsigned long long | 8字节                     | %llu |

### sizeof
作用：返回传入参数在内存中占的字节数
```c
int main(){
	int num=1;
	printf("%zu",sizeof(num));
	printf("%zu",sizeof(int)); //输出4，对应int类型类型的字节数
}
```
sizeof可用来[[数组#^39d65e|计算数组长度]]

### 浮点型（实型） float: （单精度）、double（双精度）
* C 语言用浮点类型表示小数
* float 类型的数值在末尾要加 f 或 F
* 除了普通的小数形式外，还支持科学计数法

| 数据类型 | 占内存空间 | 通配符 |
| :--- | :--- | :--- |
| float | 4字节 | %f |
| double | 8字节 | %lf |
